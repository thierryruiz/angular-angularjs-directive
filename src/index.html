Load Angular JS directive in Angular 7

The effective way to upgrade an angular JS directive in Angular new version. What is mostly undocumented is the need of downgrade the host component

  angular.module('angularJSModule', []).
      directive('angularJSDirective', function(){ return {
      template: `
        I'm an AngularJs directive!
      `  
      }}).
      directive('myApp', downgradeComponent({component: AppComponent}));
